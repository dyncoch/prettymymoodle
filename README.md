# Pretty my Moodle

This is supposed to generate a pretty HTML code to be pasted to Moodle from Intellij IDEA.

# Usage

1. Open the `pretty-my-moodle`.
2. Copy your code from Intellij IDEA.
3. Click the only button that is there.
4. Paste the generated code to Moodle.

# Requirements

- Java
- Intellij IDEA
- Moodle
